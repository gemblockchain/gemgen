# gemgen
Address generator in Rust

## Install

If you have Rust: `cargo install gemgen`

## Usage

```bash
CLI address generator for Gem

Usage: gemgen [OPTIONS]

Options:
  -c, --count <COUNT>        [default: 1]
      --chain-id <CHAIN_ID>  [default: G]
  -h, --help                 Print help
  -V, --version              Print version
```
